## Project Overview
This Power BI project provides a deep-dive analysis of revenue and performance metrics for a hotel chain. The dashboard is designed to help stakeholders monitor business health, optimize pricing strategies, and understand booking behaviors across different cities and property categories.

## Key Performance Indicators (KPIs)
The dashboard tracks and visualizes the following essential hospitality metrics:
* **Total Revenue:** Overall earnings across all properties.
* **Occupancy %:** Percentage of successful room bookings relative to capacity.
* **ADR (Average Daily Rate):** The average income earned per occupied room.
* **RevPAR (Revenue Per Available Room):** A key efficiency metric (ADR √ó Occupancy %).
* **Realization %:** The ratio of check-ins to the total number of bookings.
* **DSRN (Daily Sellable Room Nights):** Average number of rooms available for sale per day.

## Dashboard Features
* **Revenue Trends:** Visualized by city (Mumbai, Bangalore, Hyderabad, Delhi) and room category (Luxury vs. Business).
* **Temporal Analysis:** Weekly revenue and occupancy trends to identify seasonal spikes.
* **Operational Insights:** Performance comparison between Weekends and Weekdays.
* **Booking Channels:** Analysis of revenue generated through various platforms (Direct, Online, etc.).
* **Dynamic Filtering:** Slicers for City, Room Class, Month, and Week number for granular exploration.

## Tech Stack
* **Tool:** Power BI Desktop
* **Data Cleaning:** Power Query (ETL process)
* **Calculations:** DAX (Data Analysis Expressions) for custom measures and KPIs.
* **Visuals:** Native Power BI charts and custom visual integrations.

3.  **Interact:** Use the slicer panel on the left to filter the data by specific regions or timeframes.
4.  **Analyze:** Hover over data points to see detailed tooltips and insights.
